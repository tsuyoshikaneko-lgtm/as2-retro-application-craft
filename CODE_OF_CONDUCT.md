# 行動規範

このリポジトリは、古い Flash / ActionScript 2.0 制作の知見を、現代の AI エージェントと一緒に再利用するための実験テンプレートです。

Issue、Pull Request、レビュー、議論では、次を守ってください。

- 人ではなく、再現手順、設計、コード、文書、検証結果について議論する。
- 互換性、ライセンス、文化的表現に関する指摘は、できるだけ具体的な根拠や代替案と一緒に書く。
- ハラスメント、差別的な表現、脅迫、個人攻撃、意図的な荒らしを行わない。
- 問題のある AA、文言、素材、公開方法を見つけた場合は、削除または差し替えが判断できる形で報告する。

メンテナは、この行動規範に反する投稿、Issue、Pull Request、コメントを編集・非表示・削除することがあります。
